# 📲 Target Solution APK: Step-by-Step Guide

To get the **Target Integrated Solution** app on your Android phone, you need to generate the APK file from the source code provided in this project.

### ⏱ Time Required: 3-5 minutes

---

## 🚀 1. Setup Your Computer
You need to have **Flutter** installed. If you don't have it, download it from [flutter.dev](https://docs.flutter.dev/get-started/install/windows).

## 🛠️ 2. Run the Build Command
Open your terminal (Command Prompt, PowerShell, or Terminal) and run these exact commands one by one:

1.  **Go to the app folder:**
    ```bash
    cd ethio_telecom_app
    ```

2.  **Download project dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Build the APK file:**
    ```bash
    flutter build apk --release
    ```

---

## 📂 3. Where is your APK file?
Once the command finishes successfully, your APK file will be located at:
**`ethio_telecom_app/build/app/outputs/flutter-apk/app-release.apk`**

---

## 📲 4. Install it on your Phone
1.  **Transfer:** Send the `app-release.apk` file to your phone.
2.  **Open:** Locate the file on your phone and tap to open it.
3.  **Permissions:** Your phone may ask for permission to "Install unknown apps." Go to **Settings** and click **Allow**.
4.  **Launch:** Open the **Target Solution** app and start exploring!
