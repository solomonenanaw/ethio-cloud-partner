# Target Integrated Solution Mobile App

This is the official Flutter mobile application for **Target Integrated Solution Trading PLC**, showcasing their comprehensive technology services and their partnership as an Ethio Telecom Cloud Reseller.

## Features
- **Company Dashboard**: Highlighting Target Integrated Solution's core services.
- **Cloud Services Catalog**: Detailed view of cloud hosting, email, and backup services provided via Ethio Telecom partnership.
- **Direct Contact**: Click-to-call and click-to-email functionality for Target support.
- **Software Solutions**: Showcasing custom systems like Clinic and Hospital Management.
- **Mobile First Design**: Optimized for Android and iOS.

## Getting Started

1. Ensure you have Flutter installed.
2. Navigate to this directory: `cd ethio_telecom_app`
3. Install dependencies: `flutter pub get`
4. Run the app: `flutter run`

## Technical Details
- Framework: Flutter
- Language: Dart
- State Management: StatefulWidget (Native)
- Theme: Material 3
- Branding: Target Solution Brand Colors (Blue #005EB8 and Orange #F7941D)
